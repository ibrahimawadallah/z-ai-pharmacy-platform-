# 🧪 دليل اختبار منصة الصيدلة الذكية

## 🌐 **اختبار المنصة الإنتاجية**

### **الرابط:**
- **https://z-ai-pharmacy-platform.vercel.app**

### **المميزات التي يجب اختبارها:**

#### 1. **🔍 البحث عن الأدوية**
```bash
# اختبر من الواجهة:
1. اذهب إلى صفحة البحث
2. ابحث عن: "paracetamol"
3. تحقق من النتائج
4. جرب: "amoxicillin", "metformin"
```

#### 2. **🏥 المعلومات الطبية**
```bash
# صفحات المعلومات:
- /safety (السلامة الدوائية)
- /interactions (التفاعلات)
- /pregnancy (فئات الحمل)
- /dosage (الجرعات)
```

#### 3. **🤖 الذكاء الاصطناعي**
```bash
# دردشة الذكاء الاصطناعي:
1. ابحث عن الزر العائم في الزاوية اليمنى السفلية
2. اضغط عليه (دائرة زرقاء أيقونة دماغ)
3. جرب الأسئلة:
   - "What are the side effects of paracetamol?"
   - "Is aspirin safe in pregnancy?"
   - "What is the renal adjustment for metformin?"
```

#### 4. **📱 التوافق مع الموبايل**
```bash
# من الموبايل:
1. افتح الرابط في متصفح الموبايل
2. تحقق من الواجهة
3. جرب البحث والدردشة
```

## 🚀 **إعداد سيرفر Ollama للخصوصية التامة**

### **الخيار 1: سيرفر سحابي (موصى به)**

#### **DigitalOcean (5$ شهرياً):**
```bash
# 1. إنشاء Droplet
- Ubuntu 22.04
- 2GB RAM (أو 4GB أفضل)
- $5/شهر

# 2. الاتصال بالسيرفر
ssh root@your-server-ip

# 3. تشغيل السكربت
bash <(curl -s https://raw.githubusercontent.com/your-repo/main/scripts/quick-ollama-setup.sh)
```

#### **AWS EC2 (10$ شهرياً):**
```bash
# 1. إنشاء EC2 Instance
- Ubuntu Server 22.04
- t2.micro أو t3.small
- $10/شهر

# 2. نفس الخطوات السابقة
```

### **الخيار 2: سيرفر محلي (للمستشفيات)**

#### **المتطلبات:**
- 💻 جهاز قوي (8GB RAM على الأقل)
- 🌐 شبكة داخلية
- 🔐 جدار حماية

#### **الإعداد:**
```bash
# على جهاز لينكس
sudo apt update
bash scripts/quick-ollama-setup.sh
```
