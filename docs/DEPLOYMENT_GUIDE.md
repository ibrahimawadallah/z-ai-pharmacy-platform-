# دليل نشر منصة الذكاء الاصطناعي الطبي

## 🌐 كيفية جعل المستخدمين يستفيدون من النظام

### الخيار 1: النشر على سيرفر سحابي (موصى به)

#### الخطوات:
1. **نشر التطبيق على Vercel/Railway**
```bash
# تثبيت Vercel CLI
npm i -g vercel

# نشر المشروع
vercel --prod
```

2. **إعداد Ollama على السيرفر**
```bash
# على سيرفر Linux
curl -fsSL https://ollama.ai/install.sh | sh
ollama serve
ollama pull cniongolo/biomistral:latest
```

3. **إعدادات البيئة**
```env
OLLAMA_HOST=http://localhost:11434
OLLAMA_MODEL=cniongolo/biomistral:latest
```

#### المزايا:
- ✅ جميع المستخدمين يصلون عبر الإنترنت
- ✅ تحديثات مركزية
- ✅ صيانة سهلة
- ✅ نسخ احتياطي تلقائي

---

### الخيار 2: شبكة محلية (مستشفى/صيدلية)

#### الإعداد:
1. **سيرفر محلي**
```bash
# جهاز قوي في الشبكة المحلية
# تثبيت Ollama والمنصة
# إعدادات الشبكة الداخلية
```

2. **الوصول من الأجهزة**
```bash
# المستخدمون يصلون عبر:
http://192.168.1.100:3000  # مثال
```

#### المزايا:
- 🔒 خصوصية تامة للبيانات
- 🏥 مناسب للمستشفيات
- ⚡ سرعة شبكة محلية
- 💰 لا تكاليف سحابية

---

### الخيار 3: نسخة محلية للمستخدم النهائي

#### حزمة التثبيت:
1. **إنشاء ملف تثبيت**
```bash
# ملف setup.exe أو .dmg
# يتضمن:
# - التطبيق المترجم
# - Ollama مدمج
# - النموذج biomistral
# - تعليمات التثبيت
```

2. **البرنامج النصي للتثبيت**
```powershell

# 1. تثبيت Node.js
# 2. تثبيت Ollama
# 3. تحميل النموذج
# 4. تشغيل التطبيق
```

#### المزايا:
- 💻 يعمل بدون إنترنت
- 🔐 خصوصية مطلقة
- 📱 مناسب للأفراد
- 🏠 استخدام منزلي
