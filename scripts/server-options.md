# 🖥️ خيارات سيرفر Ollama

## 🏆 **الخيار 1: DigitalOcean (موصى به للمبتدئين)**

### **المواصفات:**
- 💰 **$5/شهر** (الخيار الأساسي)
- 💾 **2GB RAM, 1 CPU, 25GB SSD**
- 🌐 **عرض نطاق 1TB/شهر**
- ⚡ **سهل الإعداد**

### **الخطوات:**
1. اذهب إلى https://cloud.digitalocean.com/
2. سجل حساب جديد
3. اضغط "Create Droplet"
4. اختر:
   - **Image**: Ubuntu 22.04 LTS
   - **Plan**: Basic, $5/شهر
   - **Region**: أقرب منطقة لك
   - **Authentication**: SSH Key (أوصى)
5. اضغط "Create Droplet"
6. انتظر 30 ثانية
7. ستحصل على IP address

---

## 🏆 **الخيار 2: AWS EC2 (للمحترفين)**

### **المواصفات:**
- 💰 **$10/شهر** (t3.small)
- 💾 **2GB RAM, 2 vCPU**
- 🌐 **AWS موثوقية عالية**
- 🔧 **خيارات متقدمة**

### **الخطوات:**
1. اذهب إلى https://aws.amazon.com/ec2/
2. سجل دخول AWS
3. اضغط "Launch Instance"
4. اختر:
   - **AMI**: Ubuntu Server 22.04 LTS
   - **Instance Type**: t3.small
   - **Key Pair**: أنشئ واحفظ
5. اضبط Security Group:
   - SSH (22)
   - HTTP (80)
   - HTTPS (443)
   - Custom (11434)
6. اضغط "Launch"

## 📋 **بعد اختيار السيرفر:**

### **معلومات ستحتاجها:**
1. **IP Address** (مثل: 123.45.67.89)
2. **SSH Key** (للوصول الآمن)
3. **Username** (عادة: root)
4. **Password/Key** (للتسجيل)

### **الأدوات المطلوبة:**
- **Terminal** (Linux/Mac) أو **PuTTY** (Windows)
- **SSH Client**
- **معرفة أساسية بالأوامر**

---

## 🚀 **الخطوة التالية:**

بعد إنشاء السيرفر والحصول على IP:
1. اتصل بالسيرفر عبر SSH
2. شغل سكربت الإعداد
3. اختببر الاتصال
4. ربط بـ Vercel
